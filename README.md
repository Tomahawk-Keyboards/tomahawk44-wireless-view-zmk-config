# Tomahawk 44
A ZSA Voyager & Corne hybrid

> [!IMPORTANT]
> Changing Bluetooth profile/switching between devices: PgDn + Q/W/E/R/T \
> Erasing Bluetooth profile: PgDn + ESC \
> Bluetooth Backlight statuses: Solid 🔵 - connected, blinking 🔵 - open/advertising, blinking 🔴 - disconnected \
> Battery level: PgDn + Ins (Solid 🟢 - above 80%, solid 🟡 - above 20%, solid 🔴 - below 20%, 🟣 - not detected) 

Default bindings:
- Lower layer: PgDn + any key
- Upper layer: PgUp + any key
- Toggle RGB On/Off: PgDn + Y
- Hue Up: PgDn + U
- Hue Down: PgDn + J
- Saturation Up: PgDn + I
- Saturation Down: PgDn + K
- Brightness Up: PgDn + O
- Brightness Down: PgDn + L
- Speed Up: PgDn + P
- Speed Down: PgDn + ;
- Next Effect: PgDn + DEL
- Previous Effect: PgDn + '

> [!CAUTION]
> Physical power switch is located near the USB-C connector. \
> Left Half, Right Half \
> On <- Off, On <- Off
